    SQLite Hex-ULIDs

    4 December 2022

    While we can't natively get UUIDs or ULIDs in a SQLite database, with a bit of ingenuity we can natively generate universally unique, lexicographically sortable identifiers (ULIDs). In this post, we build a SQLite expression that can function as the default for a primary key column on a table. Our quasi-ULIDs will have the same bit structure as standard ULIDs, but encoded in base16 instead of base32. These hex-ULIDs are URL-friendly identifiers that don't expose any information about your system, but retain table sort behavior.
